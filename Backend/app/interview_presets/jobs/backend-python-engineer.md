# Python 后端工程师

## 资料来源
- 官方底座：人社部“计算机程序设计员”国家职业技能标准；Python、FastAPI、Django、PostgreSQL、Redis 官方文档。
- 面试优先级：用户上传 JD 和项目技术栈优先。

## 面试关注点
- 是否能讲清 API 设计、数据建模、异步任务、缓存、权限、安全和服务稳定性。
- 是否理解 Python 类型、协程、依赖管理、日志、测试和部署。
- 是否能用 FastAPI/Django/Flask 项目说明真实复杂度。

## 追问规则
- 优先追问：接口为什么这样设计、数据库表怎么建、Redis 用来解决什么问题、错误如何返回给前端。
- 对“会 Python”要追问工程化、性能瓶颈和线上排障。

## 评分维度
- 高分：能把业务接口、数据模型、缓存异步和部署监控讲成闭环。
- 扣分：只会脚本或 Demo，缺少后端工程意识。
