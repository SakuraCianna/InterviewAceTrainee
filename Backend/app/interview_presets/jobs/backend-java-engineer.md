# Java 后端工程师

## 资料来源
- 官方底座：人社部“计算机程序设计员”国家职业技能标准；Java/Spring 官方文档；数据库和中间件官方文档。
- 面试优先级：岗位 JD 的业务域、Spring 生态、数据库和中间件要求优先。

## 面试关注点
- 是否能讲清接口设计、事务边界、缓存一致性、并发控制、消息队列和稳定性。
- 是否理解 JVM、线程池、连接池、慢 SQL、监控告警和故障恢复。
- 是否能用项目证据说明架构取舍而不是背八股。

## 追问规则
- 优先追问：并发量是多少、瓶颈在哪里、事务怎么兜底、缓存和数据库如何一致。
- 对“我负责接口开发”要追问具体复杂度和线上问题。

## 评分维度
- 高分：基础扎实、能讲清工程取舍和线上排障。
- 扣分：只背概念，无法落到自己项目。
