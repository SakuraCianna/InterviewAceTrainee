# 面试预设知识库

这个目录存放面试官和复盘报告使用的静态预设资料。预设不是百科资料，而是为了让模型在不同岗位、专业和考试场景下问得更贴近真实面试。

## 使用原则

1. 用户上传的岗位 JD、简历、目标院校、报考专业和研究方向优先级最高。
2. 官方资料作为底座，包括人社部职业分类/国家职业技能标准、教育部研究生专业目录、国家公务员局招录公告和 IELTS 官方说明。
3. 没有单一官方职业标准的岗位，使用可靠行业框架和公开岗位要求补充，但不覆盖用户 JD。
4. 每个预设文件只保留面试可用的信息：关注点、追问规则、评分维度和扣分点。

## 覆盖范围

- 工作岗位：覆盖 250+ 个岗位面试方向，包含技术、产品、设计、运营、市场、销售、职能、财经法务、供应链、制造工程、新能源汽车、建筑土木、医药医疗、教育传媒、农业环保和公共服务等大类。这里的“方向”不是逐字复制全部职业编码，而是把真实招聘中常见的细分岗位聚合成可训练的面试入口。
- 研究生复试：覆盖 100+ 个研究生专业方向，包含教育部研究生学科目录中的主要学科门类和常见专硕方向；语言类等细分专业使用总纲覆盖多个语种，具体院校复试细则仍以用户填写的目标院校为准。
- 考公面试：按结构化面试常见题型组织，不要求用户上传资料。
- 雅思口语：按 IELTS Speaking Part 1/2/3 和官方评分维度组织，不要求用户上传资料。

## 为什么 IELTS 有 4 个预设

雅思口语只有一个产品模块，但训练中需要四个知识切片：

- 总览与评分维度：统一约束评分和报告。
- Part 1：熟悉话题、短回答、自然原因。
- Part 2：话题卡 long turn，重点是持续展开和细节。
- Part 3：抽象讨论，重点是原因、对比和解决方案。
