# 面试预设知识库

这个目录存放面试官和复盘报告使用的静态预设资料。预设不是百科资料，而是为了让模型在不同岗位、专业和考试场景下问得更贴近真实面试。

## 使用原则

1. 用户上传的岗位 JD、简历、报考专业和研究方向优先级最高。
2. 官方资料作为底座，包括人社部职业分类/国家职业技能标准、教育部研究生专业目录、国家公务员局招录公告和 IELTS 官方说明。
3. 没有单一官方职业标准的岗位，使用可靠行业框架和公开岗位要求补充，但不覆盖用户 JD。
4. 每个预设文件只保留面试可用的信息：关注点、追问规则、评分维度和扣分点。

## 主要来源

- 人社部职业分类与国家职业技能标准：https://www.mohrss.gov.cn/
- 中国就业网职业分类大典相关发布：https://chinajob.mohrss.gov.cn/
- 教育部研究生教育学科专业目录：https://www.moe.gov.cn/
- 国家公务员局中央机关及其直属机构考试录用公务员专题：https://bm.scs.gov.cn/

- IELTS 官方评分描述：https://ielts.org/organisations/ielts-for-organisations/ielts-scoring-in-detail

## 为什么 IELTS 有 4 个预设

雅思口语只有一个产品模块，但训练中需要四个知识切片：

- 总览与评分维度：统一约束评分和报告。
- Part 1：熟悉话题、短回答、自然原因。
- Part 2：话题卡 long turn，重点是持续展开和细节。
- Part 3：抽象讨论，重点是原因、对比和解决方案。
